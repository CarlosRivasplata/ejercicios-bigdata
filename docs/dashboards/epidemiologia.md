# Epidemiologia y Salud Publica

Dashboards interactivos que aplican tecnicas de visualizacion y analisis de datos al dominio de la epidemiologia.
Estos ejercicios demuestran como transformar datos de salud publica en herramientas de decision visual.

## Contexto academico

La epidemiologia cuantitativa requiere herramientas de visualizacion que permitan:

- **Vigilancia epidemiologica**: monitoreo en tiempo real de indicadores clave
- **Deteccion de brotes**: identificacion visual de anomalias en series temporales
- **Analisis espacial**: distribucion geografica de enfermedades y factores de riesgo
- **Comunicacion de riesgo**: traduccion de datos complejos en visualizaciones comprensibles para tomadores de decisiones

Estos dashboards ilustran como Python y las librerias de visualizacion modernas (Plotly, D3.js)
pueden servir como puente entre el analisis estadistico y la toma de decisiones en salud publica.
