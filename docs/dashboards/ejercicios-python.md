# Ejercicios Python: Limpieza y Analisis

Notebooks exportados como HTML interactivo que cubren el flujo completo de trabajo con datos:
desde la verificacion del entorno hasta el analisis descriptivo con tablas y graficos.

## Ejercicios interactivos

cadena completa desde datos hasta visualizacion.

## Contexto academico

Estos ejercicios cubren las etapas iniciales del ciclo de vida de los datos,
que en la practica profesional consumen entre el 60% y el 80% del tiempo de un proyecto:

| Etapa | Descripcion | Ejercicio |
|---|---|---|
| **Setup** | Verificar que el entorno funciona correctamente | 00 - Test |
| **Limpieza** | Datos crudos a datos utilizables | 02 - Limpieza |
| **Exploracion** | Entender la estructura y distribucion de los datos | 03 - Descriptivo |

### Flujo de trabajo recomendado

1. Verificar el entorno (Exercise 00)
2. Cargar datos crudos y realizar limpieza sistematica (Exercise 02)
3. Explorar con estadisticas descriptivas y tablas (Exercise 03)
4. Visualizar patrones y anomalias (conecta con modulos de ML y NLP)

### Tecnologias utilizadas

- **Pandas**: manipulacion y analisis de datos tabulares
- **NumPy**: operaciones numericas y manejo de arrays
- **Matplotlib / Plotly**: visualizacion estatica e interactiva
- **Jupyter Notebook**: entorno interactivo exportado a HTML
